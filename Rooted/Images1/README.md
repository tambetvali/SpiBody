# SpiBody — Images1 Set  
Illustrations for the Main README.md

This folder contains six images designed to accompany the SpiBody main README.md.  
Each image is placed at a specific point in the document to support pacing, clarity, and somatic flow.  
The images are not decorative: they act as structural anchors, helping the reader shift between conceptual layers.  
Their placement follows the rhythm of the text — from grounding, to coordination, to recovery, to symbolic cognition, to logic, to measurement.  
Each section below includes a visible embed, a copyable block, a full instruction block, and a placement note.
